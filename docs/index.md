# PPT 深度研究

`ppt-deep-search` 用于在生成 PPT 前完成来源理解审阅。它把论文、网页、PDF、Markdown 或仓库材料整理成 `source_understanding_review.html`，帮助人类先确认技术理解、来源证据、图片材料和证据边界。

请从三段入口阅读：

- [能力展示](reference.md)：查看 forward-test 中最新运行的交付件和覆盖情况。
- [使用方式](usage.md)：了解 Source Understanding 流程、交付物和校验命令。
- [架构概览](architecture_design.md)：直接使用仓库维护侧架构文档，说明运行契约、参考契约、评审面、交付物、forward-test 和验证层的关系。

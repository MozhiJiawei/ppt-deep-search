# RTX Spark / Agent 原生电脑深度研究输入

找到了，标题里的“全球首个 Agent 原生电脑”主要对应的是 NVIDIA RTX Spark，原始信息源不是中文媒体文章，而是 NVIDIA 官方发布 + 黄仁勋 GTC Taipei 2026 keynote。

原始来源：

1. NVIDIA 官方新闻稿：NVIDIA and Microsoft Reinvent Windows PCs for the Age of Personal AI
   - 发布时间：2026 年 5 月 31 日
   - URL: https://nvidianews.nvidia.com/news/nvidia-microsoft-windows-pcs-agents-rtx-spark
   - 这里官方说法是：“world’s first Windows PCs purpose-built for personal agents”，也就是“全球首批专为个人 Agent 打造的 Windows PC”。中文标题里的“Agent 原生电脑”是媒体转述。

2. NVIDIA RTX Spark 官方产品页
   - URL: https://www.nvidia.com/sv-se/products/rtx-spark/
   - 关键信息：RTX Spark Superchip，最高 6144 个 Blackwell RTX GPU 核心、20 核 CPU、1 PFLOP FP4 AI 性能、128GB 统一内存，面向本地 Agent、AI 开发、创作和游戏。

3. NVIDIA GTC Taipei 2026 Keynote Full Replay
   - URL: https://www.nvidia.com/en-tw/gtc/taipei/keynote/
   - 这是中文文章文末列出的参考资料，应该是它最直接的“原始信息”。

4. NVIDIA 官方博客：Microsoft Build 合作说明
   - URL: https://blogs.nvidia.com/blog/rtx-ai-garage-computex-spark-local-agents/
   - 补充 Microsoft Build 上黄仁勋和 Nadella 讨论的内容：RTX Spark、DGX Station for Windows、OpenShell、安全运行时，以及 Windows 上运行 personal/frontier agents。

5. Microsoft Build 2026 Live Blog 相关条目
   - URL: https://news.microsoft.com/build-2026-live-blog?live-feed-entry-id=4927
   - 微软侧确认：NVIDIA 和 Microsoft 合作推出面向 agentic AI 的统一加速计算栈，包括 RTX Spark Windows PCs 和 DGX Station for Windows。

一句话判断：中文标题说的“老黄重新发明 PC / 全球首个 Agent 原生电脑”，源头是 NVIDIA + Microsoft 在 GTC Taipei / Build 2026 发布 RTX Spark Windows PC 平台。官方更准确的表述是“专为 personal agents 打造的 Windows PC”，不是一个单独叫“Agent 原生电脑”的正式产品名。
